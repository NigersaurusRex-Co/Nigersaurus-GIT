Hva er github og git?




Hva er git?

Linus Torvalds designet Git, et åpen kildekode-versjonsstyringssystem, i 2005. Git er et distribuert versjonskontrollsystem, som innebærer at hver utvikleres datamaskin har tilgang til hele kodebasen og historien, noe som muliggjør enkel forgrening og sammenslåing. Over 87 prosent av utviklerne bruker Git, ifølge en Stack Overflow-utviklerundersøkelse.

Hva er github?

Vi har slått fast at Git er et versjonskontrollsystem som tilsvarer, men er overlegent enn mange andre alternativer. Hva er det med GitHub som skiller den fra konkurrentene? Selv om Git er et kommandolinjeverktøy, sentrerer alt Git rundt huben – GitHub.com – der utviklere lagrer prosjektene sine og nettverk med likesinnede andre. La oss gå gjennom noen av hovedårsakene til at nerder liker GitHub, samt litt terminologi.
